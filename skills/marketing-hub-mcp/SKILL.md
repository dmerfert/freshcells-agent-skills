# Marketing Hub MCP

Dieser Skill gibt Anweisungen zur Nutzung des freshcells Marketing Hub MCP-Servers.

## Verfügbare Tools

| Tool | Beschreibung |
|------|-------------|
| `search_vault` | Freitext-Suche über alle Glossar-Dateien |
| `get_file` | Einzelne Datei laden |
| `list_files` | Alle verfügbaren Dateien auflisten |
| `suggest` | Content-Vorschläge basierend auf einem Thema |
| `search_by_tag` | Dateien nach Tags filtern |
| `list_tags` | Alle verfügbaren Tags anzeigen |
| `get_changelog` | Letzte Änderungen im Content |
| `get_brand_voice` | Brand Voice Guide laden |
| `get_references` | Kundenreferenzen laden |
| `get_design_guide` | Design Guidelines laden |
| `get_design_tokens` | Design Tokens (Farben, Fonts, Spacing) |
| `get_glossar_overview` | Glossar-Übersicht mit allen Dateien und Beschreibungen |
| `list_assets` | Alle verfügbaren Assets (Logos, Icons) auflisten |
| `get_asset` | Asset-URL abrufen |
| `get_asset_inline` | Asset inline laden (SVG oder Base64) |
